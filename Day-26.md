# 8. Project Scheduling Techniques

## Techniques Learned

### Gantt Chart

Visual representation of activities over time.

### PERT (Program Evaluation and Review Technique)

Uses probabilistic time estimates.

#### Formula

TE = (O + 4M + P) / 6

### CPM (Critical Path Method)

Identifies the longest path in a project network.

---

## Critical Path

Critical path determines:

* Minimum project completion time
* Activities with zero float

---

## Forward Pass

Calculates:

* Earliest Start (ES)
* Earliest Finish (EF)

---

## Backward Pass

Calculates:

* Latest Start (LS)
* Latest Finish (LF)

---

# 10. Cost Monitoring Techniques

## Techniques Learned

* Budgetary control
* Earned Value Management (EVM)
* Variance analysis
* Cost baseline analysis
* Trend analysis
* Milestone monitoring

---

## EVM Metrics

### Cost Performance Index (CPI)

CPI = EV / AC

### Schedule Performance Index (SPI)

SPI = EV / PV

### Estimate at Completion (EAC)

EAC = AC + (BAC − EV)/(CPI × SPI)

### Variance at Completion (VAC)

VAC = BAC − EAC

---

# 11. Financial Evaluation Techniques

## Net Profit (NP)

NP = Total Inflow − Initial Investment

---

## Return on Investment (ROI)

ROI = (Net Profit / Investment) × 100

---

## Payback Period (PP)

Measures time required to recover investment.

---

## Discounted Cash Flow (DCF)

Considers time value of money.

### Techniques

* Net Present Value (NPV)
* Internal Rate of Return (IRR)

---

## NPV Formula

NPV = Sum(Present Value of Cash Flows) − Investment
