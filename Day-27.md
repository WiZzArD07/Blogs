# Software Engineering Learning 

## Introduction

Software Engineering is much more than just writing code. During my B.Tech 3rd Year studies, I explored how software is planned, analyzed, designed, tested, maintained, and managed systematically. This subject helped me understand how real-world software systems are developed professionally using engineering principles.

This repository/blog summarizes my overall learning in Software Engineering and the important concepts I studied throughout the course.

---

# What I Learned in Software Engineering

## 1. Introduction to Software Engineering

I learned that Software Engineering is a systematic and disciplined approach for developing software. It focuses on:

* High-quality software
* Reduced development cost
* Better maintainability
* Reliability and scalability

I also understood:

* Difference between software and hardware
* Characteristics of software
* Software crisis and its causes
* Need for software engineering
