# Day 21 - Introduction to AWS (Amazon Web Services)

## What I Learned
- AWS is a cloud computing platform provided by Amazon
- It offers services like computing, storage, databases, and networking
- AWS follows a pay-as-you-go pricing model
- It allows users to build and scale applications without managing physical infrastructure
- AWS is widely used in modern DevOps and cloud-based systems

## What is AWS
- Amazon Web Services (AWS) is a cloud platform that provides on-demand IT resources over the internet
- It allows users to access servers, storage, and databases without owning hardware
- AWS provides more than 200 services globally for building applications. 

## What is Cloud Computing
- Cloud computing means delivering IT resources over the internet
- Users can access resources anytime without managing infrastructure
- It uses a pay-as-you-go model where you pay only for what you use.

## Why AWS is Used
- No need to manage physical servers
- Easy scalability (increase or decrease resources anytime)
- Cost-effective pricing model
- High availability and reliability
- Global infrastructure with data centers worldwide

## Types of Cloud Services

### IaaS (Infrastructure as a Service)
- Provides virtual machines and storage
- Example: EC2

### PaaS (Platform as a Service)
- Provides platform for developing applications
- Example: Elastic Beanstalk

### SaaS (Software as a Service)
- Provides ready-to-use applications
- Example: Gmail, AWS tools

AWS provides a mix of IaaS, PaaS, and SaaS services.
## Core AWS Services

### EC2 (Elastic Compute Cloud)
- Provides virtual servers
- Used to run applications

### S3 (Simple Storage Service)
- Used for storing files and data

### RDS (Relational Database Service)
- Managed database service

### VPC (Virtual Private Cloud)
- Provides secure network environment

## Key Features of AWS
- Scalability: Increase resources based on demand
- Reliability: High uptime and fault tolerance
- Security: Strong security infrastructure
- Flexibility: Supports multiple programming languages and tools

## Real Use Case
A company wants to deploy a web application:
- Uses EC2 for servers
- Uses S3 for storing files
- Uses RDS for database
- Scales resources automatically based on traffic

This allows fast deployment without managing hardware.

## Summary
AWS is a leading cloud computing platform that provides on-demand services like computing, storage, and networking. It helps organizations build scalable and cost-effective applications without managing physical infrastructure.
