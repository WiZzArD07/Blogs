## 1. Introduction to Machine Learning

Machine Learning (ML) is a subset of Artificial Intelligence (AI) that enables systems to learn from data and improve performance without being explicitly programmed.

### Key Components:

* **Data (D):** Input information used for training
* **Experience (E):** Past data used to improve model
* **Performance (P):** Accuracy or success metric

### Example:

Predicting match winner based on past performance

---

## 2. Types of Machine Learning

### 2.1 Supervised Learning

* Uses labeled data
* Learns mapping: Input → Output

**Algorithms:**

* Linear Regression
* Logistic Regression
* Decision Trees
* KNN

**Example:**
Predict house price

---

### 2.2 Unsupervised Learning

* Uses unlabeled data
* Finds hidden patterns

**Algorithms:**

* K-Means Clustering
* Hierarchical Clustering

**Example:**
Customer segmentation

---

### 2.3 Reinforcement Learning

* Learns using reward and punishment

**Example:**
AI playing chess

## 8. Overfitting & Underfitting

* Model performs well on training but poor on testing

### Underfitting:

* Model fails to capture patterns

* Regularization
